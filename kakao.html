Kakao OPEN API Exercise
Explain
  • Kakao Open API 중 책 검색 기능을 활용.
  • 다음 책 서비스에서 질의어로 도서 정보를 검색.
  • Youtube 강의영상을 통해 Jqeury CDN 기능과 .ajax() 학습 후 활용.
  • 검색 창을 통해 책 이름을 Query로 도서 정보를 Request.
  • Response로 도서 제목, 도서 소개, 도서 상세 URL, 도서 판매가, 도서 표지 미리보기, 도서 판매 상태 정보를 받아 출력.
  • .append() 사용 시 검색 결과가 쌓이게 되엇 .html()을 사용.
  • try-catch문을 사용하여 유효하는 결과값이 없을 경우, 알림창 출력, 새로고침.
